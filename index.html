About Me

Hello! I consider myself to be a computer enthusiast, software craftsman, web developer, aspiring private pilot, transportation engineering hobbyist, golfer, and student. I am competent in numerous languages, both for humans and machines. Having spent the past 7 years dabbling in various fields of software development, I hope to have gained a strong level of discipline and fluency from my experience. I strive to maximize professionalism and quality by following rigorous methodologies from the software industry, such as Agile. I develop applications in a modular, testable architecture with Test-Driven Development and Continuous Integration, using proper source control (Git).

Technologies

JVM

After beginning to program with Java since 2013, I have experimented with various other JVM languages, including Kotlin and Clojure.

Python and Ruby

The backend of PreMatch.org is powered by Flask, a web microframework for Python. The PreMatch Discord Bot is powered by Ruby.

Functional Languages

I have been working with functional constructs and languages. I worked on Advent Of Code 2018 and 2019 with solutions written in Haskell. I made Zeitplan (a personal time management project) with ClojureScript.
I find value in functional concepts such as segregation of impure functions, immutability, and composition over inheritance.

Systems Languages

Occasionally, I program in C, C++, and Rust. Recently I have placed myself in a dilemma between Nim and Crystal, and I have chosen Nim for its Windows support, relative maturity, and interesting metaprogramming philosophy.

The Web

This website was made with HTML and Sass (SCSS), bundled by Jekyll. I use JavaScript and TypeScript on new PreMatch experiments, such as Virtual Bell. Virtual Bell was built using Vue, and I am experimenting with Svelte as a promising candidate for a PreMatch makeover. Although I have not used Elm in any experiments yet, I find its Model-View-Update architectural pattern to be very elegant.

Android and iOS

I maintain the PreMatch for iOS app in Swift, and I technically create Android applications while programming for the FIRST Tech Challenge.
